Chapters II–III

Resonance Foundry & exploration rails

Research nodes unlock from hunter level, hunter XP, unified points (game, battle, …), and prerequisite nodes—same progression as the Hunters game and Profile economy. First-time research grants unified points and hunter XP on the server. Some nodes use a 1h / 3h / 10h re-research cooldown after you clear them. Exploration pulse uses a 3h profile cooldown. Co-tech: 1h between new drafts, 10h between agent refines per draft.

Cooldown policy (nodes + projects)

Timers start when you clear a researched node (re-research gate), when you save a co-tech draft (1h), after agent refine on that draft (10h), or after an exploration pulse (3h).

TierDurationApplies to
1h3600sLight Ch.II tools (codec, cache, chrono, echo dampener, magnetometer, helix bus, events horizon, gallery orbit). Also: new co-tech draft spacing (profile).
3h10800sGated Ch.II nodes + geo-anchor; all Ch.III exploration rails except rule fold / core lock. Also: exploration pulse (profile).
10h36000sPoints prism, rulebook node, foundry key, rule fold, foundry core. Also: agent refine per technology draft.

Exact per-node seconds live in data/lab_progression_catalog.json (research_cooldown_sec).

Pulse: +3 activity · +1 game · +6 XP. Deep scan: +9 activity · +4 game · +20 XP.
