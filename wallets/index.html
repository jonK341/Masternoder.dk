Connect to the MN2 network

New wallets often stall at “0 connections” if bootstrap peers are missing. MasterNoder2 uses P2P port 17646 (not 5133). RPC for local apps is 9332.

  1. After first launch, open masternoder2.conf in your datadir (%APPDATA%\MasterNoder2 on Windows, ~/.masternoder2 on Linux).
  2. Ensure listen=1, port=17646, dnsseed=1, and several addnode=host:17646 lines — do not use connect=localhost (that isolates the wallet).
  3. Restart the wallet; you should see 4+ peers within a few minutes. Use the peer list below or fetch /api/mn2/network-peers.

Quick setup

  1. Download the build for your platform above (daemon for servers, Qt for desktop).
  2. Extract the archive and run masternoder2d or masternoder2-qt.
  3. Copy config/masternoder2.conf.example from this repo to your datadir and set rpcuser / rpcpassword.
  4. Wait for sync, then send MN2 to your profile deposit address to fund your in-app balance.

Windows users without WSL: use the Qt wallet zip, or run the Linux daemon inside WSL for headless RPC. See project docs MN2_DAEMON_SETUP.md on GitHub for full details.

Verify downloads

Compare the SHA256 checksum shown on each card with your file after download. Release manifests are published on GitHub releases. Never enter your wallet seed or private keys on masternoder.dk — only on official MN2 software you run locally.
