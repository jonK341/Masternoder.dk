===========================================
THEME BACKGROUND IMAGES DIRECTORY
===========================================

This directory is for theme background images.

HOW TO ADD YOUR PICTURES:
-------------------------

1. Place your image files here (JPG or PNG format)
2. Recommended size: 1920x1080 or larger
3. Recommended file size: Under 2MB

CURRENT THEME IMAGES NEEDED:
-----------------------------

- stargate_bg.jpg      (Stargate theme)
- mtg_bg.jpg           (Magic: The Gathering theme)
- warhammer40k_bg.jpg  (Warhammer 40K theme)
- halo_bg.jpg          (Halo theme)
- dnd_bg.jpg           (Dungeons & Dragons theme)
- disney_bg.jpg        (Disney theme)
- matrix_bg.jpg        (Matrix theme - optional, currently uses gradient)

QUICK START:
------------

1. Rename your image to match one of the theme names above
2. Place it in this directory
3. Run: ..\..\..\..\..\..\deploy.bat (from project root)
4. Clear browser cache: Ctrl + Shift + R
5. Switch to the theme in Settings to see your image!

EXAMPLE:
--------

If you have a space image you want for Stargate theme:
1. Rename it to: stargate_bg.jpg
2. Place it here: src/web/static/img/themes/stargate_bg.jpg
3. Upload to server
4. Done! The Stargate theme will use your image.

For more details, see: THEME_BACKGROUND_IMAGES_GUIDE.md
