# Theme Pack — Starter

Personal-use starter pack for MasterNoder UI surfaces.

## Included Presets

- Galaxy Dark: deep background, high-contrast cards, green/blue accents.
- Neon Green: commerce-ready accent layer for shop and profile CTAs.

## Suggested CSS Tokens

```css
:root {
  --mn-theme-bg: #071018;
  --mn-theme-panel: #101b2a;
  --mn-theme-accent: #00ff88;
  --mn-theme-accent-2: #39a8ff;
  --mn-theme-text: #f4f8fb;
}
```
