# Prompt Pack — Rich Video Starters

Curated prompt starters for MasterNoder generator workflows.

## Prompts

1. **Product reveal**
   - "Create a 12-second cinematic reveal of [product] with glowing UI overlays, smooth dolly movement, and a high-contrast dark studio background."

2. **Founder explainer**
   - "Generate a concise founder-style explainer for [idea], showing problem, product, and call-to-action in three visual beats."

3. **Feature loop**
   - "Show [feature] as a looping interface animation with readable state changes, subtle particles, and clear before/after contrast."

4. **Commerce teaser**
   - "Create a premium shop teaser for [offer], using neon accents, centered product card, and a final frame with a strong buy-now cue."
