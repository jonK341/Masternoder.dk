## 📋 Overview

## 🎯 Abilities

Abilities are special powers that can be used. They're separate from skills but often require certain skill levels.

### How Abilities Work

1. **Unlock** - Meet prerequisites
2. **Use** - Activate ability
3. **Upgrade** - Improve through use
4. **Master** - Reach maximum level

### Ability Types

- **Combat Abilities** - Battle-related powers
- **Magic Abilities** - Spell-based powers
- **Tech Abilities** - Technology-based powers
- **Social Abilities** - Social interaction powers
- **Creative Abilities** - Creative expression powers
- **Physical Abilities** - Physical enhancement powers

## 📊 Aggregator Resources

### What Are Aggregator Resources?

Aggregator resources track your skill usage and provide bonuses. They aggregate data from all your skill activities.

### How They Work

1. **Track** - Monitor skill usage
2. **Aggregate** - Combine data from all sources
3. **Calculate** - Determine bonuses
4. **Apply** - Apply bonuses to activities

### Resource Types

- **Total Resources** - Sum of all skill points
- **Average Resources** - Average skill level
- **Count Resources** - Number of skill uses
- **Peak Resources** - Highest skill achievement

## ❓ FAQ

### Q: How do skills relate to points?

**A:** Skills contribute to the unified point system through aggregator resources. Skill activities earn both skill points and regular points.

### Q: Can I reset my skills?

**A:** Skills can be reset, but this requires special items or events. Contact support for skill resets.

### Q: Do skills have maximum levels?

**A:** Skills have soft caps but can continue improving. Mastery is achieved at high levels.

### Q: How do abilities unlock?

**A:** Abilities unlock when you meet prerequisites, usually skill levels or achievements.
