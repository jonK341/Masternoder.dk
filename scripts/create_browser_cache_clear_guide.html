Clear Browser Cache - MasterNoder Profile Page

Method 1: Hard Refresh (Fastest)

Windows/Linux: Press Ctrl + Shift + R or Ctrl + F5

Mac: Press Cmd + Shift + R

Method 2: Clear Cache via Settings

Chrome/Edge:

  1. Press Ctrl + Shift + Delete (Windows) or Cmd + Shift + Delete (Mac)
  2. Select "Cached images and files"
  3. Time range: "All time"
  4. Click "Clear data"

Firefox:

  1. Press Ctrl + Shift + Delete
  2. Select "Cache"
  3. Time range: "Everything"
  4. Click "Clear Now"

Method 3: Incognito/Private Window

Open the profile page in an incognito/private window - this bypasses all cache:

  • Chrome: Ctrl + Shift + N
  • Firefox: Ctrl + Shift + P
  • Edge: Ctrl + Shift + N

Then navigate to: https://masternoder.dk/vidgenerator/profile

Method 4: Disable Cache (Developer Tools)

  1. Open Developer Tools (F12)
  2. Go to Network tab
  3. Check "Disable cache" checkbox
  4. Keep DevTools open and refresh the page

What Should You See After Clearing Cache?

  • ✅ Profile loads with user data
  • ✅ Statistics section shows actual numbers (not "Loading...")
  • ✅ Achievements section loads
  • ✅ Points and XP display correctly
  • ✅ No console errors
